# SatoSystem

SatoSystemは仮想通貨の自動取引ボットのプロジェクトです。このプロジェクトは、取引所APIの使用、トレード戦略の実装、リスク管理など、自動取引ボットの基本的な要素を実装したサンプルを提供します。

## プロジェクト概要

このプロジェクトは以下のクラスで構成されています。

- `Bot`: メインのボットクラスで、取引所、取引戦略、ポートフォリオ、リスク管理を組み合わせて自動トレードを実行します。
  - **処理概要**: ボットの初期化、取引戦略の実行、リスク管理の適用、取引所との通信を行います。
  - **主要メソッド**:
    - `start()`: ボットの実行を開始します。
    - `stop()`: ボットの実行を停止します。
    - `execute_trade()`: トレード戦略を実行します。
- `BybitExchange`: Bybit取引所の操作を管理するクラスです。
  - **処理概要**: Bybit APIを使用して、注文の作成、キャンセル、口座情報の取得を行います。
  - **主要メソッド**:
    - `place_order(order)`: 注文を作成します。
    - `cancel_order(order_id)`: 注文をキャンセルします。
    - `get_balance()`: 口座残高を取得します。
- `Config`: ボットの設定情報を格納するクラスで、APIキー、トレードの間隔、戦略の選択などの設定を保持します。
  - **処理概要**: 設定ファイルの読み込みと設定値の提供を行います。
  - **主要メソッド**:
    - `load_config(file_path)`: 設定ファイルを読み込みます。
    - `get(key)`: 指定された設定値を取得します。
- `Event`: イベント駆動型の設計をサポートするクラスです。
  - **処理概要**: イベントの登録、発火、リスナーの管理を行います。
  - **主要メソッド**:
    - `register(event_name, listener)`: イベントリスナーを登録します。
    - `emit(event_name, data)`: イベントを発火します。
- `Logger`: ログ情報を収集し、記録するためのクラスです。
  - **処理概要**: ログメッセージのフォーマットと保存を行います。
  - **主要メソッド**:
    - `info(message)`: 情報ログを記録します。
    - `error(message)`: エラーログを記録します。
- `Order`: 注文情報を格納するクラスで、注文のタイプ、通貨ペア、数量、価格などの情報を保持します。
  - **処理概要**: 注文データの構造化と操作を行います。
  - **主要メソッド**:
    - `to_dict()`: 注文情報を辞書形式に変換します。
    - `validate()`: 注文データを検証します。
- `RiskManagement`: リスク管理戦略を実装するクラスです。
  - **処理概要**: ポジションサイズの計算、損失制限、リスク許容範囲の適用を行います。
  - **主要メソッド**:
    - `calculate_position_size(balance, risk)`: ポジションサイズを計算します。
    - `apply_risk_limits(order)`: リスク制限を適用します。
- `TradingStrategy`: トレーディング戦略を表現するクラスで、エントリーポイントと出口ポイントを決定します。
  - **処理概要**: 市場データを分析し、売買シグナルを生成します。
  - **主要メソッド**:
    - `generate_signal(data)`: 売買シグナルを生成します。
    - `backtest(data)`: 戦略のバックテストを実行します。

## インストール

このプロジェクトをローカル環境にクローンするには、次のコマンドを使用します。

```bash
git clone https://github.com/do3-1979/satosystem.git
```

## 使い方

1. リポジトリをクローンします。
2. 必要なライブラリをインストールします。
   ```bash
   pip install -r requirements.txt
   ```
3. `config.ini` ファイルを編集し、取引所のAPIキーとAPIシークレットを設定します。
4. ボットを実行します。
   ```bash
   python bot.py
   ```

## ドキュメンテーション

プロジェクト内の各クラスにはGoogleスタイルのドキュメンテーションが適用されています。  
ドキュメントを生成するには以下のコマンドを使用してください（例: `pdoc` を使用）:

```bash
pdoc --html --output-dir docs .
```

詳細な情報は各ソースファイル内のドキュメントを参照してください。
