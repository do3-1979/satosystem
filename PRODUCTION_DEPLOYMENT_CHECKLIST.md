## 🔍 トラブルシューティング

### エラー: "socket timeout"
```bash
解決策：ネットワーク接続確認、サーバ再起動
リトライ：自動実装（最大 3 回）
```

### エラー: "insufficient balance"
```bash
原因：ポジションが多すぎる、またはドローダウン
対応：risk_percentage を下げて再実行
```

### エラー: "no such column: symbol"
```bash
原因：キャッシュスキーマが古い
解決策：
  rm ohlcv_data/ohlcv_cache.db
  システム再起動で自動再生成
```

### 0 トレード問題
```bash
原因：市場条件、またはシグナル生成エラー
診断：
  1. config.ini の market 設定を確認
  2. logs/backtest_summary_*.json をチェック
  3. 回帰テスト実行：python test/regression_test_suite.py
```
