## 所見
1. 四半期ごとに MFE/ATR と MAE/ATR の分布がシフト (Q2 低下, Q3 上昇) → 市場レジーム変動を反映。
2. PnL 大幅貢献は Q3 (トレンド強期) に集中し Q2/Q4 はドローダウン。
3. 推奨k2 はトレンド強期に上昇 (2.0 → 2.7) し弱期に低下 (2.6 → 2.0) する自然な動き。
4. k3 は 1.7→1.8→1.9 と徐々に引き上げられ FALSE_BREAK 許容を減らす方向。
5. 現在は閾値が取引ロジックに未統合なため適応効果は分析のみに留まる。

## 改善アイデア (次ステップ)
- 統合案A (軽量): 直近 n 取引の MAE/ATR が推奨k3超過多発時は ENTRY抑制フラグを一定バー有効。
- 統合案B (拡張): 予測モデル (MFE期待値 vs MAE期待値) で TREND確度が低い場合 PVO閾値を +X% 引き上げ。
- 統合案C (EXIT): 保持中ポジの実現中MAE/ATR が推奨k3 を超えた場合早期損切り候補に昇格。

## 推奨アクション
1. EXIT戦略優先: Q2/Q4 のドローダウン抑制 (max_hold_bars / ADX連動 / ボラ縮小EXIT)。
2. 分類活用パス1: ENTRY前に "最近のFALSE_BREAK割合 > 30%" なら Donchianブレイクを一度遅延評価。
3. 2025年へ拡張: 同一手法で 2025年Q1-Q4 を取得しレジーム遷移パターン検出。

## KPI再定義 (分析軸)
- 四半期 TREND比率 > 40%
- FALSE_BREAK 比率 < 30%
- TREND四半期 (PF > 1.2 かつ PnL>0) 継続率 >= 50%
