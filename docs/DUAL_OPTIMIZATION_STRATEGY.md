# 勝ちQ利益向上 × 負けQ損失回避 両立戦略

## 概要

Phase 0 の分析から、勝ちQ（4つ）と負けQ（4つ）には明確に異なる環境特性と失敗パターンがある。これを活用して、**勝ちQを保護しながら負けQを改善する戦略**を提案する。

---

## 1. 現状分析

### 1.1 勝ちQuarter の特性

| 指標 | 値 | 特徴 |
|------|-----|------|
| **数** | 4つ (Q1 2024, Q3 2024, Q4 2024, Q4 2025) | 全体の50% |
| **合計PnL** | +$724.39 | 大幅な利益 |
| **平均WR** | 92.3% | 非常に高い |
| **平均PF** | 1.61 | 良好 |
| **市場環境** | 上昇トレンド | Donchianが有効 |

**勝ちQの強み:**
- エントリー判断が極めて正確 (WR 92.3%)
- 環境対応が良好 (トレンド環境では WR 100%)
- エグジットロジックが優秀 (PF 1.61)

### 1.2 負けQuarter の特性

| 指標 | 値 | 特徴 |
|------|-----|------|
| **数** | 4つ (Q2 2024, Q1 2025, Q2 2025, Q3 2025) | 全体の50% |
| **合計PnL** | -$480.84 | 大幅な損失 |
| **平均WR** | 29.0% | 著しく低い |
| **平均PF** | 0.72 | 悪い |
| **市場環境** | レンジ、ボラティリティ高 | トレンド不明 |

**負けQの問題点:**
- WR が著しく低い (29.0% vs 92.3%)
- 環境対応が失敗 (トレンド不明環境でも同じロジック)
- ドローダウンが大きい (パターン 2)

### 1.3 負けQuarter の詳細分類

#### グループ A: エントリー判断の失敗 (WR 低)
- **Q1 2025**: WR 7.9%, PnL -$143.83 ← 最悪
- **Q2 2025**: WR 12.8%, PnL -$169.05 ← 最大損失
- **原因**: 非トレンド環境でのエントリー
- **対応**: エントリーフィルター強化 (トレンド強度判定)
- **勝ちQへの影響**: ✅ **非常に低い** (勝ちQはWR 92.3%)

#### グループ B: SL幅の不適切さ (WR 高いが損失大)
- **Q3 2025**: WR 62.0%, PnL -$133.87, DD $409 ← ドローダウン最大
- **原因**: 高ボラティリティ環境での不適切なSL設定
- **対応**: 環境別ポジションサイジング / SL幅調整
- **勝ちQへの影響**: ⚠️ **要注意** (調整方法によっては WR 低下の可能性)

#### グループ C: 軽微な損失の積み重ね
- **Q2 2024**: WR 33.3%, PnL -$33.09 ← 軽度
- **原因**: トレード数が多く、小さな負けが積み重なる
- **対応**: トレード選別 (高品質エントリーのみ取る)
- **勝ちQへの影響**: ✅ **非常に低い** (むしろ利益向上)

## 2. 改善策の提案

### 提案 1: エントリーフィルター強化（優先度: **最高**）

**対象**: グループ A (Q1 2025 WR 7.9%, Q2 2025 WR 12.8%)

#### 戦略

現在のエントリー条件:
```
1. Donchian 20期間 上抜け
2. PVO > 0 (上昇シグナル)
3. 過去2時間の値動きチェック
```

改善案:
```
1. Donchian 20期間 上抜け
2. PVO > 0 (上昇シグナル)
3. 過去2時間の値動きチェック
4. + ADX >= 25 (トレンド強度フィルター)  ← 追加
```

#### 期待効果

- Q1 2025, Q2 2025 での不適切なエントリーをフィルター
- 期待改善: 両Quarter で WR 7-13% → 30-40% へ上昇
- 推定PnL改善: -$143 + -$169 = -$312 → -$100 程度 (改善額 +$200)

#### 勝ちQへの影響

✅ **非常に低い (むしろ保護効果)**

理由:
- 勝ちQの環境 = 上昇トレンド (ADX 高い)
- 勝ちQ のWR = 92.3% (既に高い)
- ADX >= 25 フィルターは、トレンド環境でのエントリーを選別
- 勝ちQのエントリーは大部分が ADX >= 25 のはず

**リスク評価**: ⭐ 低い

---

### 提案 2: ボラティリティベース ポジションサイジング（優先度: **高**）

**対象**: グループ B (Q3 2025 WR 62%, DD $409)

#### 戦略

現在のSL設定:
```python
stop_offset = volatility * 2.0
```

改善案:
```python
# 環境別のSTOP_RANGEを設定
if volatility > HIGH_VOL_THRESHOLD:  # 例: $2000以上
    stop_range = 1.5  # SLを狭める = ポジションサイズを減らす
elif volatility < LOW_VOL_THRESHOLD:  # 例: $500未満
    stop_range = 2.5  # SLを広げる = ポジションサイズを増やす
else:
    stop_range = 2.0  # 通常

stop_offset = volatility * stop_range
```

#### 期待効果

- Q3 2025: DD $409 → $250 程度に削減
- WR は変わらないが、損失単位が小さくなる
- PnL改善: -$133.87 → -$50 程度 (改善額 +$80)

#### 勝ちQへの影響

⚠️ **中程度の注意が必要**

理由:
- Q1 2024: volatility 低い → stop_range 2.5 → SL広がる → 利益増加？
- Q4 2025: volatility 低い → stop_range 2.5 → SL広がる → OK
- ただし SL が広がると、損失トレードの損失が大きくなる可能性

**リスク評価**: ⭐⭐ 中程度 (テストが必須)

**テスト方法**: 
1. Q1 2024, Q3 2024, Q4 2024 で検証
2. PnL 悪化がないか確認
3. WR 低下がないか確認

### 提案 3: エントリー品質フィルター（優先度: **中高**）

**対象**: グループ C (Q2 2024 WR 33.3%, PnL -$33)

#### 戦略

現在: すべての Donchian 上抜け + PVO > 0 をエントリー

改善案: PVO の大きさで選別
```python
if pvo > PVO_HIGH_THRESHOLD:  # 例: 0.5以上
    # 強いシグナル、エントリー
    proceed_entry()
elif pvo > 0:
    # 弱いシグナル、スキップ
    skip_entry()
```

#### 期待効果

- 高品質エントリーのみを取得
- Q2 2024: 45トレード → 30トレード程度に削減
- WR は変わらないが、トレード数削減で損失削減
- PnL改善: -$33 → +$10 程度 (改善額 +$40)

#### 勝ちQへの影響

✅ **非常に低い (むしろ利益向上)**

理由:
- 勝ちQのトレード数 39.2/Q 平均
- 高品質フィルターで 35トレード程度に削減
- WR 92.3% なので、高品質フィルター後も WR は維持
- むしろ利益率が上がる可能性

**リスク評価**: ⭐ 低い
