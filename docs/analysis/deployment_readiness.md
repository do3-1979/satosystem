## 📞 サポート体制

### エスカレーション手順

```
事象発生
  ↓
daily_monitor.py でアラート検出
  ↓
【Level 1】: 日次監視者が確認（15分以内）
  ├─ Win Rate低下 → market_analysis.md確認
  ├─ PF低下 → 直近3日の取引内容確認
  └─ 問題なし → ログのみ記録
  ↓
【Level 2】: パラメータ微調整必要か判定（1時間以内）
  ├─ 必要 → プリセット1/2/3から選択して切り替え
  ├─ 様子見 → 翌日継続監視
  └─ 重大 → Level 3へエスカレート
  ↓
【Level 3】: 緊急対応（30分以内）
  ├─ Win Rate 30%以下 × 2日連続 → Phase 1 停止
  ├─ ドローダウン -$300以上 → ポジションサイズ50%削減
  ├─ システムバグ → ロールバック + 調査
  └─ その他重大 → 手動トレード切り替え
```

### 連絡先・責任者

| 役割 | 担当者 | 連絡先 | 対応時間 |
|------|--------|--------|---------|
| 日次監視 | Ops Team | Slack #trading-alerts | 08:00-18:00 |
| 技術サポート | Dev Lead | developer@example.com | 08:00-22:00 |
| 最高責任者 | PM | pm@example.com | 24時間 |
