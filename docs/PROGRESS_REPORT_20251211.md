## ❌ 失敗フェーズの教訓

### Phase 2: Box市場 Donchian逆張り戦略 
**期間**: 2025-12-11 中盤～後期  
**実装内容**:
- Donchian Channel (20期) 極値タッチで逆張り
- Bollinger Bands 2σ外接触を補助シグナル
- ポジションサイズ: 60% (Donchian) / 40% (BB)

**バックテスト結果**:
```
Phase 1.5: +$266.49
↓ (Phase 2 適用)
Phase 2: +$156.97
────────────────
悪化: -$109 ❌
```

**問題点**:
| Q | Phase 1.5 | Phase 2 | 差 | 根因 |
|---|-----------|---------|-----|------|
| Q1 2024 | +$216 | +$620 | +$404 | 偶然 |
| Q4 2024 | +$77 | -$115 | -$192 | 逆張り損失 |
| Q4 2025 | +$321 | -$13 | **-$334** | 強トレンドで逆張り最悪 |

**根本原因分析**:
- Donchian High/Low = ブレイクアウト兆候 ≠ 反発ポイント
- 逆張り戦略は **BTC相場（トレンド継続性高）に不適切**
- 理論的誤解: "Box市場 = 逆張り" の単純化が失敗

**判定**: ❌ **廃棄** → 理論的に不適切

**ドキュメント**: `PHASE2_FAILURE_ANALYSIS.md` に詳細分析記載

## 🔮 Phase 3 実装予定

### 戦略概要

```
現状 (Phase 1.5):
  if ADX < 20: skip
  elif 20 ≤ ADX < 25: 50% position
  else: 100% position

改善 (Phase 3 案):
  adx_1h = get_adx(1h)
  adx_4h = get_adx(4h)
  
  if adx_1h < 20 AND adx_4h < 25: skip
  elif 20 ≤ adx_1h < 25 AND adx_4h < 20: skip  # 4h がBox
  elif adx_1h < 25 AND adx_4h < 20: skip       # 4h がBox優先
  elif 20 ≤ adx_1h < 25: 50%
  elif adx_1h ≥ 25 AND adx_4h ≥ 25: 100%
  else: 75%  # 1h 強でも 4h Weak なら中程度
```

### 期待改善
- 2025 Q1-Q3 で False Positive削減
- より確実なトレンド判定
- ノイズフィルタリング効果
