# バックテスト可視化ガイド

## 概要

`visualizer.py` は、`bot_run.sh` で実行したバックテスト結果を、インタラクティブなHTMLグラフとして可視化します。

## 実行方法

### 1. バックテスト実行

```bash
cd src
./bot_run.sh
```

これにより、`src/logs/` ディレクトリにバックテストログが保存されます。

- JSON形式: `YYYYMMDDHHMMSS.json` (メインログ)
- テキスト形式: `latest_backtest.log` (参考用)

### 2. グラフ生成

```bash
cd src
python3 visualizer.py
```

実行後、最新のバックテストログを自動検出し、インタラクティブなHTMLグラフを生成します。

### 3. グラフ表示

生成されたHTMLファイルをブラウザで開きます：

```
report/backtest_visualization.html
```

## 出力ファイル

- **ファイル名**: `report/backtest_visualization.html`
- **ファイルサイズ**: 約 150-200 KB
- **オープン方法**: ブラウザで直接開く（ダブルクリック またはドラッグ&ドロップ）

## グラフの構成

### Row 1: 価格チャート

- **2時間足ローソク足**: メインのキャンドルスティックチャート
- **Donchian High/Low**: ドンチャンチャネルの上限・下限（緑/赤の破線）
- **PSAR**: パラボリックSAR（オレンジのマーカー）
- **Keltner Channel**: ケルトナーチャネル（青の点線）
- **ポジション区間**: 背景色でハイライト
  - 淡緑: BUYポジション保有中
  - 淡赤: SELLポジション保有中
- **ENTRY/ADD/EXIT マーカー**:
  - 🔼 **ENTRY** (ライムグリーン): ポジション開始
  - 🔴 **ADD** (イエロー): ポジション追加（ピラミッディング）
  - 🔽 **EXIT** (赤): ポジション終了
- **損切値 (Stop)**: オレンジの点線

### Row 2: ボリューム

- **Volume Bar**: 取引高の棒グラフ（スチールブルー）

### Row 3: テクニカル指標

- **Volatility**: ボラティリティ（紫）
- **PVO**: パーセンテージボリュームオシレータ（シアン）
- **ADX**: 平均方向指数（紺）
- **ATR**: 平均真の値（茶色、デフォルト非表示）

### Row 4: 累積損益 (PnL)

- **Total PnL**: 累積損益ライン（濃青の面積グラフ）
- **ゼロライン**: グレーの破線

## 対話機能

### ズーム・パン

- **ズーム**: マウスホイール または ドラッグで範囲選択
- **パン**: 手のひらアイコンツールを使用してスクロール
- **リセット**: ホームアイコンをクリック

### 表示・非表示切替

- **凡例をクリック**: 指標の表示/非表示を切り替え
- **Shift+クリック**: 特定指標のみ表示/非表示

### ホバー情報

- マウスカーソルを乗せると、詳細な情報が表示されます
- 日時、価格、ポジション情報など

### ダウンロード

- **カメラアイコン**: 現在の表示状態をPNGで出力

## 設定

### config.ini での期間設定

`src/config.ini` の以下の項目でバックテスト期間を指定します：

```ini
[Period]
start_time = 2025/11/1 0:00
end_time = 2025/11/30 23:59
```

- `start_time`: テスト開始時刻
- `end_time`: テスト終了時刻
- ログに含まれるデータがない場合、グラフは生成されません

### 計算用のルックバック期間

デフォルトで 20日間の過去データを遡って計算しますが、グラフには指定期間のデータのみを表示します。

## トラブルシューティング

### 「No log files」エラーが出る

- `bot_run.sh` を実行していることを確認
- `src/logs/` ディレクトリにJSONファイルがあるか確認

```bash
ls -l src/logs/*.json
```

### グラフが空または データが少ない

- `config.ini` の期間設定を確認
- バックテスト期間とグラフの表示期間が一致しているか確認

### ブラウザで開けない

- HTMLファイルのパス: `/home/satoshi/work/satosystem/report/backtest_visualization.html`
- ファイルをコピーして別の場所で開く
- 別のブラウザを試す
