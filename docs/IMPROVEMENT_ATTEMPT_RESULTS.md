
| **勝ちQ合計** | +$724.39 | +$434.02 | -$290.37 | **-40.1%** |

**原因分析:**

勝ちQのトレードの大部分が **PVO < 0.5 の「弱いシグナル」で構成されている**。

つまり:
- 勝ちQ (特にQ1 2024, Q4 2024, Q4 2025) では「弱いPVOシグナルでも成功している」
- PVO フィルターは逆効果になった
- このアプローチは不適切

**結論:** SKIPPED

### 提案 2: ボラティリティベース動的SL - ❌ FAILED

**実装内容:**
```python
volatility = get_volatility()

if volatility > 2000:      # 高ボラティリティ環境
    stop_range = 1.5       # SLを狭める
elif volatility < 500:     # 低ボラティリティ環境
    stop_range = 2.5       # SLを広げる
else:
    stop_range = 2.0       # 通常

stop_offset = volatility * stop_range
```

**結果:**

| 指標 | Phase 0 | 実装後 | 変化 | 変化率 |
|------|---------|--------|------|--------|
| **トータルPnL** | +$593.23 | +$240.57 | -$352.66 | **-59.4%** |
| **勝ちQ合計** | +$724.39 | +$434.02 | -$290.37 | **-40.1%** |

**悪化したQuarter:**
- Q1 2024: +$460.17 → +$215.92 (-$244.25, -53.1%)
- Q4 2024: +$142.53 → +$76.80 (-$65.73, -46.1%)
- Q4 2025: +$348.68 → +$271.24 (-$77.44, -22.2%)

**改善したQuarter:**
- Q3 2025: -$133.87 → -$114.01 (+$19.86, +14.8%)
- Q2 2025: -$169.05 → -$151.21 (+$17.84, +10.6%)
- Q1 2025: -$143.83 → -$141.13 (+$2.70, +1.9%)

**原因分析:**

勝ちQのQ1 2024, Q4 2024, Q4 2025 は **低ボラティリティ環境** で発生している。

- stop_range = 2.5 に設定されてSLが広がる
- SLが広いと負けトレードの損失が大きくなる
- 結果的に勝ちQが損傷

つまり:
- **負けQの改善 > 勝ちQの悪化** というトレードオフが発生
- 単純なボラティリティ調整では不十分

**結論:** SKIPPED
