# Phase 0 vs Phase 1 詳細分析レポート

**レポート日**: 2025-12-11  
**分析対象**: Phase 0 (フィルタなし) vs Phase 1 (シンプル ADX<25 フィルタ)  
**期間**: 8四半期 (2024 Q1～2025 Q4)

---

## 📊 Executive Summary

### 結論：ADX<25 フィルタは**意図した効果を達成できず**

| 指標 | Phase 0 | Phase 1 | 評価 |
|------|---------|---------|------|
| **累積PnL** | +$593.23 | +$31.00 | ❌ 94.8% 悪化 |
| **2024年度** | +$691.29 | +$46.38 | ❌ 93.3% 悪化 |
| **2025年度** | -$98.07 | -$15.38 | ✅ 84.3% 改善 |
| **負け四半期の損失** | -$479.84 | -$293.60 | ✅ $186 削減 |

**判定**: ADX フィルタは 2025年度の損失を抑制したが、2024年度の利益を著しく損なったため、**全体では大幅悪化**。

## 💡 根本原因分析

### ADX<25 フィルタの問題点

**問題 1: 閾値が不適切**
```
理想的なシナリオ (想定):
  ADX < 20 = Box → ノイズ多い → フィルタすべき
  ADX ≥ 25 = 強トレンド → クリア → エントリーすべき

現実 (結果):
  ADX 20-25 の境界は頻繁に遷移
  Q1 2024, Q3 2024: ADX が 20-25 で浮遊 → ほぼ全取引フィルタ
```

**問題 2: Donchian ブレイクとの相性悪**
```
Donchian 20-period ブレイク = 既に市場精査済みシグナル
  → 低ボラティリティ期は自動的にシグナル少ない
  → ADX フィルタ = 過度な二重フィルタ

結果: 利益を上げる可能性のあるシグナルまで排除
```

**問題 3: 市場レジーム判断が粗い**
```
ADX 値のみで市場判断 → 限界あり

例: Q1 2024
  - ADX が 18-26 で変動
  - 「この時間帯は ADX=24 だからスキップ」
  - → 1 本先の Donchian ブレイクで +$10 の利益を落とす
  - → 累積すると -$298
```

## 📋 負けトレード分析（特に重要）

### 負けトレードの抑制効果

| 指標 | Phase 0 | Phase 1 | 改善度 |
|------|---------|---------|--------|
| **負け四半期数** | 4/8 | 5/8 | ❌ 1Q 増加 |
| **負け四半期の累計損失** | -$479.84 | -$293.60 | ✅ $186.24 (38.8% 削減) |
| **平均損失 (負けQあたり)** | -$119.96 | -$58.72 | ✅ 51% 改善 |

**判定**: 
- ✅ **負けトレードは削減**（1Q増えても損失額は削減）
- ❌ **負けQが増加**（Q4 2024 が新規で負けに）

### 具体的な負けトレード減少メカニズム

**Q1 2025 の例**:
```
Phase 0: -$143.83 (7.9% 勝率)
  → 低ADX環境で大量の Box trap トレード
  → 38 取引のうち 35 が損切り
  
Phase 1: -$26.29 (67.7% 勝率)
  → ADX フィルタが大量の Box trap 排除
  → エントリー数削減 → 勝率向上
  → 数少ない取引で損失最小化
```

**結論**: ADX フィルタは**ノイズトレード排除には効果的**だが、利益期間でも効果を発揮してしまう。

## 🚨 意図と結果のギャップ

### 当初の仮説
```
「2025年度の損失 (-$98) は Box 市場増加が原因。
  ADX < 25 フィルタで ノイズ取引を排除すれば、
  2025年度改善 → 全体も改善する」
```

### 実際の結果
```
✅ 2025年度: 改善成功 (+$82.68)
  しかし
❌ 2024年度: 大悪化 (-$644.91)

全体では: +$593 → +$31 (-$562)
```

### 根本的な誤りの発見
```
【誤った認識】
  Phase 0 = Donchian + PVO で十分なシグナル品質
  + ADX フィルタ = さらに品質向上

【正しい認識】
  Phase 0 = 既に最適化されたシグナル
  + ADX フィルタ = 信号品質を下げる（特にトレンド市場）
  
  2025年度の損失は市場環境変化 (Entry の問題ではなく Exit の問題)
```

## 🎯 最終判定

### ADX<25 フィルタの評価

| 評価軸 | 結果 | 判定 |
|-------|------|------|
| **全体 PnL** | +$593 → +$31 | ❌ 失敗 |
| **2024年度** | +$691 → +$46 | ❌ 大失敗 |
| **2025年度** | -$98 → -$15 | ✅ 成功 |
| **負けトレード抑制** | ✅ 38.8% 削減 | ✅ 成功 |
| **勝率改善（2025年度）** | +20-60 pt | ✅ 成功 |
| **利益QP生存** | 4Q → 3Q | ❌ 悪化 |

### 理由付き結論

**フェーズ 1 は採用すべきでない**

理由:
1. **全体では大幅悪化** (-$562, -94.8%)
2. **利益四半期が大幅削減** (4Q → 3Q)
3. 2025年度の改善は**本質的な解決ではなく単なる取引削減**
4. **トレンド市場での完全な機能停止**
